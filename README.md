# Food Delivery
    FoodDelivery — это REST API приложение для сервиса доставки еды с поддержкой 
    различных ролей пользователей: клиент, курьер и администратор. Проект реализует базовую 
    бизнес-логику онлайн-заказов, управления продуктами и отзывами.
